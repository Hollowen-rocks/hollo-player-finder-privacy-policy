Privacy Policy for Hollo's Player-Finder

Introduction

Hollo's Player-Finder is committed to protecting your privacy. This privacy policy explains how we handle the information we receive and what data we collect when you use our Chrome extension.

Data Collection

Our extension does not collect, store, or share any personal information, such as your name, email address, or browsing history.

Permissions

The extension requires the following permissions to function properly:

  • activeTab: Allows the extension to access the currently active tab to interact with the Roblox website.
  • host permissions: Specifically for https://www.roblox.com/* to fetch user profile data, game server information, and avatar thumbnails.

Data Usage

The data fetched from Roblox APIs (such as user profiles, game servers, and avatar thumbnails) is only used within the extension to provide the intended functionality and is not stored or shared.

Changes to This Privacy Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.
